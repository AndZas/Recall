# IDEAS

## Search
- Unified search (speech + vision + metadata).
- Exact phrase search.
- Semantic search.
- OCR search.
- Group results by match type.
- Search returns multiple moments, not one file.

## Moment-first
- Detect interesting segments automatically.
- Highlight them on the timeline.
- Export only selected moment.
- Mark used segments.

## AI
- Whisper transcription.
- Frame analysis every N seconds.
- Object detection.
- Scene descriptions.
- Automatic tags.
- Reanalyze library button.

## Library
- Watch folders.
- Auto-index new videos.
- Automatic sorting.
- Processing priorities.

## Editing
- Preview.
- Trim.
- Export clip.
- Editable subtitles.
- Optional profanity handling.

## Future ideas
- Image support.
- Smart collections.
- Quality scoring for photos.
- Timeline-based media browsing.
