# Recall

### *Find moments, not files.*

> Recall is a local AI-powered media library that understands your
> content, automatically organizes it, and helps you instantly find,
> preview, and reuse the exact moments you need.

> **Project Status:** 🟡 Planning

------------------------------------------------------------------------

# Why Recall?

Modern creators accumulate hundreds of hours of recordings. Finding one
specific moment often takes longer than editing it.

Recall is designed to solve this problem by automatically indexing
media, understanding its content, and making every important moment
searchable.

------------------------------------------------------------------------

## Media Library

-   Automatic indexing

## Editing

-   Preview clips
-   Timeline navigation
-   Clip export

------------------------------------------------------------------------

# Philosophy

Recall is **not** another video editor.

Recall is **not** another file explorer.

Recall is an intelligent media library built around **moments instead of
files**.

Every feature should save creators time.

AI is a tool, not the goal.
