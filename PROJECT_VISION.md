# Recall

## Mission

Recall helps creators instantly find, preview and reuse the exact moment they need from large media libraries.

## Core philosophy

- The primary entity is a **Moment**, not a video file.
- AI exists to save time.
- Every feature must reduce manual work.
- Local-first whenever possible.

## Typical workflow

1. New video appears.
2. Recall indexes it.
3. Speech, frames and metadata are analyzed.
4. Interesting moments are detected.
5. User searches naturally.
6. User previews and exports only the needed clip.

## MVP

- Library indexing
- Folder monitoring
- Speech transcription
- Frame analysis
- Unified search
- Preview
- Clip export

## Future

- Subtitle editor
- Automatic highlight detection
- Smart collections
